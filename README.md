# Portfolio Website

## Overview
This project is a personal portfolio website designed to showcase projects, skills, and design work.

The website focuses on clean UI, responsive layout, and clear presentation of content.

---

## Features
- Responsive design (mobile & desktop)
- Project showcase section
- Clean and minimal UI
- Fast and lightweight

- JavaScript
